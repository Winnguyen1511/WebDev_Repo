# Create Domain name:  
- Register a domain name: [DOT.CF](http://www.dot.cf/en/index.html?lang=en)  
- Create the forward URL: [Freenom](https://my.freenom.com/)  
- Add DNS of Freenom: 80.80.80.80 and 80.80.81.81
